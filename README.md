# Teslo API
1. Clonar proyecto
2. ```yarn install```
3. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```
4. Cambiar variables de entorno
5. Levantar base de datos en docker
```
docker-compose up -d
```

6. Levantar: ```yarn start:dev```
7. Ejecutar Seed
```
http://localhost:3000/api/seed
```
